
Валютный телеграм-бот

Данный телеграм-бот сообщает актуальный курс валют(EUR, USD, CNY),
достаточно отправить ему любое сообщение.
Разновидность валют и их кол-во можно изменять в файле cbrClient.go.
Найти его можно в телеграме по @learning_test_telegram_bot.
Это мой первый проект(микро-проект), так что не судите строго :) Удачи!
